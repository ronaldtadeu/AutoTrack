Documentação - Controle de Automóveis

Configurando o Postman

Para utilizar as rotas da API de controle de automóveis utilizando o Postman, siga os passos abaixo:

  1. Abra o Postman e crie uma nova requisição.
  2. Selecione o método HTTP desejado (GET, POST, PUT, DELETE) para a rota desejada.
  3. Insira a URL da rota desejada. Por exemplo:
    4. http://localhost:3000/automoveis
  4. Se necessário, adicione os parâmetros de consulta na seção "Params" ou "Query Params".
  5. Se necessário, adicione o corpo da requisição na seção "Body". Certifique-se de selecionar o formato adequado (por exemplo, JSON).
  6. Clique em "Send" para enviar a requisição.

Exemplos de Uso

Abaixo estão alguns exemplos de uso das rotas da API:

Cadastrar um Automóvel (POST)

Método: POST

URL: http://localhost:3000/automoveis

Corpo:

{ "placa": "ABC1234", "cor": "Preto", "marca": "Fiat" }

Listar Automóveis (GET)

Método: GET

URL: http://localhost:3000/automoveis

Recuperar um Automóvel por ID (GET)

Método: GET

URL: http://localhost:3000/automoveis/{id}

Substitua {id} pelo ID real do automóvel desejado.

Atualizar um Automóvel por ID (PUT)

Método: PUT

URL: http://localhost:3000/automoveis/{id}

Substitua {id} pelo ID real do automóvel que deseja atualizar.

Corpo: (dados a serem atualizados)

{ "placa": "XYZ5678", "cor": "Vermelho", "marca": "Toyota" }

Excluir um Automóvel por ID (DELETE)

Método: DELETE

URL: http://localhost:3000/automoveis/{id}

Substitua {id} pelo ID real do automóvel que deseja excluir.

Cadastrar um Motorista (POST)

Método: POST

URL: http://localhost:3000/motoristas

Corpo:

{ "nome": "João da Silva" }

Listar Motoristas (GET)

Método: GET

URL: http://localhost:3000/motoristas

Recuperar um Motorista por ID (GET)

Método: GET

URL: http://localhost:3000/motoristas/{id}

Substitua {id} pelo ID real do motorista desejado.

Atualizar um Motorista por ID (PUT)

Método: PUT

URL: http://localhost:3000/motoristas/{id}

Substitua {id} pelo ID real do motorista que deseja atualizar.

Corpo: (dados a serem atualizados)

{ "nome": "Maria da Silva" }

Excluir um Motorista por ID (DELETE)

Método: DELETE

URL: http://localhost:3000/motoristas/{id}

Substitua {id} pelo ID real do motorista que deseja excluir.

Criar Registro de Utilização de Automóvel (POST)

Método: POST

URL: http://localhost:3000/utilizacoes

Corpo:

{ "automovelId": 1, "motoristaId": 1, "dataInicio": "2024-03-14T10:00:00", "motivoUtilizacao": "Viagem de negócios" }

Finalizar Utilização de Automóvel por ID (PUT)

Método: PUT

URL: http://localhost:3000/utilizacoes/{id}/finalizar

Substitua {id} pelo ID real do registro de utilização que deseja finalizar.

Listar Registros de Utilização de Automóvel (GET)

Método: GET

URL: http://localhost:3000/utilizacoes

Observações

  • Substitua {id} na URL pelos IDs reais dos automóveis ou motoristas.
  • Para rotas que exigem autenticação, consulte a documentação adicional.
  • Para mais detalhes sobre as rotas disponíveis, consulte o código-fonte do projeto.
